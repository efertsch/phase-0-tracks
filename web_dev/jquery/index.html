jQuery in Brief: A Tutorial for the Novice Programmer

What is jQuery?

jQuery is a lightweight Javascript library that:

  • Simplifies Javascript Programming
  • Is easy to learn and interact with
  • Allows developers to add dynamic features to web pages
  • Includes many useful plug-ins

What languages should prospective learners be familiar with?

  • HTML
  • CSS
  • Javascript

Getting Started

jQuery can be downloaded from The jQuery Website or can be included from a Content Delivery Network (CDN) such as Google or Microsoft.

If jQuery is downloaded locally, the decompressed version should be used for development and debugging.

The jQuery file can be referenced in an HTML file and incorporated directly or kept in a separate javascript file within the same directory and externally linked to the HTML file (recommended).

The document ready event

Using the document ready event prevents jQuery code from running before the page has rendered allowing the developer to place the script file in the head tag of the HTML file as opposed to within the body. This is also applicable for the Javascript file!

Example Syntax:

$(document).ready(function(){ jQuery methods go here... });

Basic Syntax

The basic syntax is as follows:

$(selector).action()
  • The '$' defines/accesses jQuery
  • The selector finds HTML elements
  • The 'action' represents the jquery action to be performed

CSS selectors are used to access elements but some are custom to jQuery

jQuery Events Methods

jQuery is designed to respond to HTML events

HTML events are described as different visitor actions that a webpage can respond to.

An event is a precise moment when some action happens. The term fires/fired is often used to describe that moment.

Syntax for setting events:

  1. Assign click event
    2. $(“selector”).someEvent();
  2. Define what happens when the event fires (pass the function to an event)
    3. $("p").click(function(){ // action goes here!!});
